# BeaconBoard

A Developer Tools dashboard that turns build logs into a launch checklist.

## Setup

```bash
npm install
npm run dev
npm test
```

Open http://localhost:3000 and upload a JSON build log.

## Built with Codex and GPT-5.6

Codex helped scaffold the repository, write parser tests, and iterate on the dashboard layout. GPT-5.6 is used in the app to summarize build failures into a release checklist.
